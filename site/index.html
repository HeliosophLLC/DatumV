people.sql ✕
SELECT
LET classes = models.yolox_s(a.file),
image_crop(file, c.value.bbox)
FROM datasets.coco_val2017 a
CROSS JOIN unnest(classes) c
WHERE c.value.label = 'person'
LIMIT 100
✓ 100 rows27.6s

Not just analysis — generation

Paint with prompts. Ask an LLM. Same SELECT.

Diffusion models and local LLMs are just functions here. Generate a column of images, or run a language model over every row — no serving stack, no Python, nothing leaves the box.

dream.sql
SELECT p.text AS prompt,
models.sdxl_turbo(p.text) AS image
FROM prompts p;
✓ 8 images3.1s

A whole table of images from a whole table of prompts. Swap in dreamshaper_hyper or realistic_vision_hyper for a different look.

ask.sql
SELECT a.title,
models.llama31_8b(
'One-line summary: ' || a.body) AS tldr
FROM articles a
LIMIT 50;
✓ 50 rows12.4s

Call any local LLM by name — llama31_8b, mistral_7b, qwen25_coder_7b — and swap models without touching the query.
